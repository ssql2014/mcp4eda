GTKWave MCP Server GTKWave MCP 服务器

GTKWave MCP Server provides programmatic access to GTKWave's powerful waveform viewing and analysis capabilities. It enables automated waveform analysis, format conversion, and visualization generation for EDA workflows. GTKWave MCP 服务器提供对 GTKWave 强大的波形查看和分析功能的程序化访问。它支持自动化波形分析、格式转换和 EDA 工作流程的可视化生成。

Available Tools 可用工具

gtkwave_open

Open waveform files in GTKWave viewer 在 GTKWave 查看器中打开波形文件

gtkwave_convert

Convert between waveform formats 在波形格式之间转换

gtkwave_extract_signals

Extract signal hierarchies from waveforms 从波形中提取信号层次

gtkwave_analyze_timing

Perform timing measurements and analysis 执行时序测量和分析

gtkwave_generate_script

Generate TCL scripts for automation 生成用于自动化的 TCL 脚本

gtkwave_capture_screenshot

Capture waveform display screenshots 捕获波形显示截图

Example: Convert VCD to FST 示例：将 VCD 转换为 FST

{ "tool": "gtkwave_convert", "arguments": { "inputFile": "large_sim.vcd", "outputFile": "compressed.fst", "format": "fst", "compress": true } }
